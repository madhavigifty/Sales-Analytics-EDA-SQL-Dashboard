# Sales-Analytics-EDA-SQL-Dashboard

## Project Overview
This project performs Exploratory Data Analysis (EDA) and Business Intelligence on a sales dataset using Python, MySQL, and Excel.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- MySQL
- Excel

## Project Steps
1. Data Cleaning
2. Exploratory Data Analysis
3. SQL Business Queries
4. Dashboard Creation
5. Key Business Insights

## Key Insights
- Electronics generated the highest revenue.
- Female and Male customers contributed almost equally.
- Adults generated the highest sales.
- Patna was the top-performing city.
- Laptop was the best-selling product.

## Dashboard
The Excel dashboard summarizes KPIs and sales trends.
